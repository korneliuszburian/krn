# KRN Compact Runtime

This directory is written by project-local Codex compact hooks.

The files here are runtime continuity metadata, not source truth. After a compact event, read:

1. newest user message,
2. `AGENTS.md`,
3. `.krn/compact/latest-postcompact.md`,
4. `.krn/compact/latest-checkpoint.md`,
5. `docs/memory/INDEX.md`.

Do not store secrets, raw transcripts, credentials, or private customer data here.
